Format

Min  Hour Day  Mon  Weekday

{: .-setup}

*    *    *    *    *  command to be executed
┬    ┬    ┬    ┬    ┬
│    │    │    │    └─  Weekday  (0=Sun .. 6=Sat)
│    │    │    └──────  Month    (1..12)
│    │    └───────────  Day      (1..31)
│    └────────────────  Hour     (0..23)
└─────────────────────  Minute   (0..59)

Example Description

*/15 * * * * every 15 mins

0 0 * * 0 every Sunday midnight
